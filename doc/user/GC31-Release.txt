Some notes on what is new in GC 3.1

MAJOR NEW FEATURES

1. Compare Mode
   * Across Summary, Ride, QA, CP, Histogram, Metric charts
   * Compare Athletes, Seasons, Rides, Intervals
   * Easy and intuitive user interface

2. Merge Activities
3. Share Dialog
